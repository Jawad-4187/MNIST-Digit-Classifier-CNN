# Handwritten Digit Recognition using CNN (MNIST)

This project implements a Convolutional Neural Network (CNN) to recognize handwritten digits from the MNIST dataset with a final test accuracy of **98.99%**.

## 📊 Project Results
* **Final Test Accuracy:** 98.99%
* **Training Accuracy:** 99.49%
* **Validation Accuracy:** 99.03%
